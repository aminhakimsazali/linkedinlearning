what is GitHub?
------
~~GitHub~~ is a platform for *hosting* and sharing code **repositories.** It allows developers to collaborate on projects, `track` changes, and manage versions of their code. It uses Git, a distributed version control system, to keep track of every change made to the codebase.
What is Version Control?
